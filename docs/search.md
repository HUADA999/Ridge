## Ridge Search
### Use
1. Open Ridge Search
  - **On Web**: Open <https://app.ridge.dev/> in your web browser
  - **On Obsidian**: Click the *Ridge search* icon ðŸ”Ž on the [Ribbon](https://help.obsidian.md/User+interface/Workspace/Ribbon) or Search for *Ridge: Search* in the [Command Palette](https://help.obsidian.md/Plugins/Command+palette)
  - **On Emacs**: Run `M-x ridge <user-query>`
2. Query using natural language to find relevant entries from your knowledge base. Use [query filters](./advanced.md#query-filters) to limit entries to search
